byte_ranges(env, size) click to toggle source

Parses the “Range:” header, if present, into an array of Range objects. Returns nil if the header is missing or syntactically invalid. Returns an empty array if none of the ranges are satisfiable. 

     # File lib/rack/utils.rb, line 278
278:     def byte_ranges(env, size)
279:       # See <http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html#sec14.35>
280:       http_range = env['HTTP_RANGE']
281:       return nil unless http_range
282:       ranges = []
283:       http_range.split(/,\s*/).each do |range_spec|
284:         matches = range_spec.match(/bytes=(\d*)-(\d*)/)
285:         return nil  unless matches
286:         r0,r1 = matches[1], matches[2]
287:         if r0.empty?
288:           return nil  if r1.empty?
289:           # suffix-byte-range-spec, represents trailing suffix of file
290:           r0 = [size - r1.to_i, 0].max
291:           r1 = size - 1
292:         else
293:           r0 = r0.to_i
294:           if r1.empty?
295:             r1 = size - 1
296:           else
297:             r1 = r1.to_i
298:             return nil  if r1 < r0  # backwards range is syntactically invalid
299:             r1 = size-1  if r1 >= size
300:           end
301:         end
302:         ranges << (r0..r1)  if r0 <= r1
303:       end
304:       ranges
305:     end

rfc2822(time) click to toggle source

Modified version of stdlib time.rb Time#rfc2822 to use ’%d-%b-%Y’ instead of ’% %b %Y’. It assumes that the time is in GMT to comply to the RFC 2109.

NOTE: I’m not sure the RFC says it requires GMT, but is ambigous enough that I’m certain someone implemented only that option. Do not use %a and %b from Time.strptime, it would use localized names for weekday and month. 

     # File lib/rack/utils.rb, line 268

270:       mon = Time::RFC2822_MONTH_NAME[time.mon - 1]
271:       time.strftime("#{wday}, %d-#{mon}-%Y %H:%M:%S GMT")
272:     end

     # File lib/rack/utils.rb, line 157
157:     def select_best_encoding(available_encodings, accept_encoding)
158:       # http://www.w3.org/Protocols/rfc2616/rfc2616-sec14.html
159: 
160:       expanded_accept_encoding =
161:         accept_encoding.map { |m, q|
162:           if m == "*"
163:             (available_encodings - accept_encoding.map { |m2, _| m2 }).map { |m2| [m2, q] }
164:           else
165:             [[m, q]]
166:           end
167:         }.inject([]) { |mem, list|
168:           mem + list
169:         }
170: 
171:       encoding_candidates = expanded_accept_encoding.sort_by { |_, q| -q }.map { |m, _| m }
172: 
173:       unless encoding_candidates.include?("identity")
174:         encoding_candidates.push("identity")
175:       end
176: 
177:       expanded_accept_encoding.find_all { |m, q|
178:         q == 0.0
179:       }.each { |m, _|
180:         encoding_candidates.delete(m)
181:       }
182: 
183:       return (encoding_candidates & available_encodings)[0]
184:     end
set_cookie_header!(header, key, value) click to toggle source 
     # File lib/rack/utils.rb, line 187
187:     def set_cookie_header!(header, key, value)
188:       case value
189:       when Hash
190:         domain  = "; domain="  + value[:domain] if value[:domain]
191:         path    = "; path="    + value[:path]   if value[:path]
192:         # According to RFC 2109, we need dashes here.
193:         # N.B.: cgi.rb uses spaces...
194:         expires = "; expires=" +
195:           rfc2822(value[:expires].clone.gmtime) if value[:expires]
196:         secure = "; secure"  if value[:secure]
197:         httponly = "; HttpOnly" if value[:httponly]
198:         value = value[:value]
199:       end
200:       value = [value] unless Array === value
201:       cookie = escape(key) + "=" +
202:         value.map { |v| escape v }.join("&") +
203:         "#{domain}#{path}#{expires}#{secure}#{httponly}"
204: 
205:       case header["Set-Cookie"]
206:       when nil, ''
207:         header["Set-Cookie"] = cookie
208:       when String
209:         header["Set-Cookie"] = [header["Set-Cookie"], cookie].join("\n")
210:       when Array
211:         header["Set-Cookie"] = (header["Set-Cookie"] + [cookie]).join("\n")
212:       end
213: 
214:       nil
215:     end
